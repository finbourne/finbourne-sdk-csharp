# Finbourne.Sdk.Lusid.Model.DrawingEvent

Mandatory partial bond redemption (DRAW) where the issuer lottery-selects specific bonds for early redemption.  The affected face amount (AFFB) is pre-determined externally from the vendor notification and supplied on the event.
> **Inherits from:** [InstrumentEvent](InstrumentEvent.md)

## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **PaymentDate** | **DateTimeOffset** | Optional | Date the cash actually flows for the drawn portion of the holding. |
| **EffectiveDate** | **DateTimeOffset** | Optional | Lottery Date (&#x3D; Record Date). Holdings are snapshotted at the close of this date to determine the affected balance. |
| **AffectedAmount** | **decimal** | Required | Affected face amount (AFFB) — the lottery-selected portion of the holding that is redeemed. Must be strictly positive. |
| **PricePerUnit** | **decimal?** | Optional | Redemption price per unit (OFFR / 100). Clean price convention.  Optional: AFFB is typically known before the issuer publishes OFFR, so a null price is permitted on upsert. |
| **Currency** | **string** | Required | Settlement currency for the redemption. |
