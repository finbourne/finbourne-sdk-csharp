# Finbourne.Sdk.Lusid.Model.CdsCreditEvent

Definition of a credit event for credit default swap (CDS) instruments.

## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **EffectiveDate** | **DateTimeOffset** | Optional | The date of the credit default - i.e. date on which the debt issuer defaulted on its repayment obligation. |
| **AuctionDate** | **DateTimeOffset?** | Optional | The date of the credit event auction - i.e. date on which the defaulted debt is sold via auction, and a recovery rate determined. |
