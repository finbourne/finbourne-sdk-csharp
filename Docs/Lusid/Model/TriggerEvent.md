# Finbourne.Sdk.Lusid.Model.TriggerEvent

Definition of a trigger event.  This is an event that occurs on transformation of an option instrument being  triggered by a barrier/touch price level being hit by the underlying asset.
> **Inherits from:** [InstrumentEvent](InstrumentEvent.md)

## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **Level** | **decimal** | Required | The underlying price level that triggers the event |
| **TriggerType** | **string** | Required | The type of the trigger; valid options are Knock-In, Knock-Out, Touch or No-Touch |
| **TriggerDirection** | **string** | Required | The direction of the trigger; valid options are Up and Down |
| **TriggerDate** | **DateTimeOffset** | Required | The date the trigger happens at. |
| **MaturityDate** | **DateTimeOffset** | Required | The date the trigger takes effect. |
| **InstrumentEventType** | **string** | Required | The Type of Event.
