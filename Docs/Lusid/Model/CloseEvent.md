# Finbourne.Sdk.Lusid.Model.CloseEvent

The termination of an instrument.  In some cases termination can happen over a range of dates e.g. american option exercise.  In most cases the startDate == endDate
> **Inherits from:** [InstrumentEvent](InstrumentEvent.md)

## Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| **StartDate** | **DateTimeOffset** | Optional | The first date on which the instrument could close |
| **EndDate** | **DateTimeOffset** | Optional | The last date on which the instrument could close |
